# Linkly

Linkly - Shorten links, track real-time analytics, and manage your profile in style.

## About
A simple and efficient URL shortener that gives you full control. See who clicks your links, where they are from, and handle everything from a clean dashboard.

## Key Features
- **Shorten Links**: Paste a long URL, get a short one. Simple.
- **Custom Links**: Make it your own (e.g., `linkly.com/mysite`).
- **Expiry**: Set links to expire automatically.
- **Analytics**: See click counts and countries in real-time.
- **Profile**: Upload your avatar and manage your account.

## Built With
- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB

## How to Run

1. **Clone it**
   ```bash
   git clone <repository-url>
   cd Linkly
   ```

2. **Setup Backend**
   ```bash
   cd BACKEND
   npm install
   npm run dev
   ```

3. **Setup Frontend**
   ```bash
   cd FRONTEND
   npm install
   npm run dev
   ```

4. **Visit**: `http://localhost:5173`
